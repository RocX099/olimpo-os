Modo

Perfil Misión Herramientas Personalidad Seguridad Avanzado Enviar

1. Perfil

Para adaptar el lenguaje, la profundidad y el tipo de propuesta.

2. Misión

Aquí definimos para qué existe tu OS. Si esto no está claro, todo lo demás sobra.

3. Herramientas y módulos

No todo el mundo necesita todo. Marca lo que podría aportar valor y escribe lo que no quieres.

4. Personalidad

El OS debe tener una presencia propia: tono, criterio y forma de acompañarte.

5. Seguridad y límites

Esto evita crear un sistema útil pero peligroso.

6. Avanzado

Solo para afinar arquitectura, despliegue, modelos e integraciones.
